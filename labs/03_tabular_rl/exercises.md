# Tabular RL Exercises

1. Run value iteration on deterministic GridWorld.
2. Run policy iteration and compare values against value iteration.
3. Change the discount factor and describe the value changes.
4. Run Monte Carlo control and inspect the greedy policy.
5. Run Q-learning and inspect the greedy policy.
6. Run SARSA and compare the greedy policy against Q-learning.
7. Run Expected SARSA and compare average returns.
8. Move Q-learning to Gymnasium FrozenLake.
