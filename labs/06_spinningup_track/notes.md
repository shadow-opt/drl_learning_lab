# Spinning Up Track Notes

This module is an index, not a vendored copy of Spinning Up.

## Use Pattern

- Read the Spinning Up explanation for the algorithm.
- Read the local mapping note.
- Run the local core demo or training command.
- Compare equations with local modules under `src/drl_lab/algorithms/`.
- Write differences in the algorithm report.
