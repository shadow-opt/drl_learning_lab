# 06 Spinning Up Track Lab Guide

## 前置阅读

先读 `course/06_spinningup_track.md`。本 lab 是外部阅读对照层，不是 Spinning Up 源码副本。

## 实验目标

- 列出每个算法的 Spinning Up mapping。
- 选择一个算法，说明外部教材、中文讲义、lab demo 和 `src` 实现如何对应。
- 保持 license 和来源信息完整。

## 代码入口

```bash
conda run -n drl-lab python labs/06_spinningup_track/code/list_mappings.py
```

Mapping 文件在 `labs/06_spinningup_track/*/spinningup_mapping.md`。

## 提交产物

- 完成 `exercises.md`。
- 填写 `report.md`。
- 不复制大段上游源码。

## 常见坑

- 把 Spinning Up 当成本仓库唯一学习内容。
- 只读外部材料，不回到本仓库跑 demo 和写 report。
- 忘记保留来源与 MIT license。
