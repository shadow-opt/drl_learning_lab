# Spinning Up Track Exercises

1. For each mapping file, identify the local module that implements the core
   equation.
2. Write one paragraph explaining how the local PyTorch style differs from
   Spinning Up's original implementation style.
3. For VPG, PPO, DDPG, TD3, and SAC, run the local smoke command and record the
   exported ONNX files.
4. For TRPO, run the math demo and explain how the KL-scaled step relates to
   PPO clipping.
