# PyTorch Foundations Exercises

1. Create a tensor shape notebook or script covering reshape, view, permute, and broadcasting.
2. Implement an MLP classifier with explicit shape assertions.
3. Add checkpoint save and load.
4. Add an eval-only script.
5. Export the model to ONNX.
6. Compare PyTorch and ONNXRuntime outputs.

Acceptance criteria:

- `pytest` passes.
- Exported ONNX model passes consistency tolerance.
- The report includes seed, device, dtype, shape, and dependency versions.
