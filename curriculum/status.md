# Curriculum Status

本文件记录学习章节当前是否适合小白独立推进。

## 状态定义

- `beginner-ready`：默认只会 Python 基础；章节提供讲解、walkthrough、预期输出、提示、参考答案和复盘模板。
- `usable`：可运行、可学习，但部分解释仍较压缩，需要主动查资料。
- `advanced`：建议完成 00-04 后再进入。
- `reference-track`：外部资料映射，不是独立训练章节。
- `engineering-track`：工程专题，建议配合算法章节学习。
