# Resources

## DRL 主参考

- OpenAI Spinning Up: https://spinningup.openai.com/
- Source repository: https://github.com/openai/spinningup
- License: MIT，来源说明保存在 `external/spinningup/`

Spinning Up 用作 policy-gradient 和 actor-critic 算法的概念主线。不要把它的旧实现风格当成本仓库的主要代码风格。
