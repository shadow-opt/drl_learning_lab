# Resources

## Core DRL Reference

- OpenAI Spinning Up: https://spinningup.openai.com/
- Source repository: https://github.com/openai/spinningup
- License: MIT, documented in `external/spinningup/`

Use Spinning Up as the main conceptual sequence for policy-gradient and
actor-critic algorithms. Do not copy its old implementation style as the main
code style for this lab.

## PyTorch and Export

- PyTorch documentation: https://docs.pytorch.org/
- PyTorch ONNX exporter docs: https://docs.pytorch.org/docs/stable/onnx_export.html
- ONNXRuntime Python API: https://onnxruntime.ai/docs/api/python/api_summary.html

## Suggested Study Order

1. PyTorch tensor and autograd basics
2. Supervised learning training loops
3. RL math and tabular RL
4. DQN
5. VPG and PPO
6. DDPG, TD3, and SAC
7. Export and deployment validation
